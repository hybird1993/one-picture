## 目录：  

├─public
│  └─img    // 背景设置所用的图片
├─src
│  ├─assets
│  │  ├─image   // 图片资源
│  │  └─style   // 公共样式
│  ├─components
│  │  ├─alarm-list    // 告警信息
│  │  ├─alarm-overview    // 告警视图总览
│  │  ├─general-power    // 综治力量
│  │  ├─global-index    // 全局指数
│  │  ├─latest-news    // 最新资讯
│  │  ├─map-setting    // 模块显示控制及背景设置
│  │  ├─one-picture   // 地图引入
│  │  ├─option-index    // 分项指数
│  │  ├─people-house   // 人口与房屋
│  │  ├─popup-window    // 点击后才会打开的窗口 eg:告警详情、人员详情、楼栋信息、通行记录等
│  │  └─special-people   // 特殊人群
│  ├─request    // 请求封装
│  └─utils    // 公共配置及方法
└─.env *  一些环境配置
